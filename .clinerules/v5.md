# Coding Assistance Rules

## Common Assumptions

- **Target Tasks**: Coding assistance, refactoring, debugging, development-related documentation
- **Language**: Follow user instructions and input language
- **Rule Priority**: System > Workspace common rules > This file
- **Completion Policy**: Do not stop midway; persist until request fulfilled
- **Instruction Priority**: Follow user specifications over defaults
- **Response Style**: Concise, direct, no verbose responses

## Tool Usage

### File Operations
- **Reading**: `read_text_file` single files, `read_multiple_files` batch
- **Directory**: `list_directory`, `directory_tree`, `search_files`
- **Editing**: `edit_file` precision, `write_file` complete rewrites
- **Creation**: `create_directory` for structures

### Memory Bank
- **Selective Pulling**: Surgical reads only
- **Reading**: `fast_read_file(path="/absolute/path")`
- **Updating**: `fast_edit_block` for synchronization
- **Writing**: `fast_write_file` for archiving only

### Search Operations
- **Basic**: `search` fast patterns
- **Advanced**: `advanced-search` with filters
- **Analysis**: `count-matches` frequency
- **Discovery**: `list-files`, `list-file-types`

### Parallel Execution
- **Safe**: Execute independent operations in parallel
- **Sequential**: File modifications always sequential

### Web/Browser Tools
- **Searching**: Proactively search for latest specs, breaking changes, risky bugs
- **Preview**: E2E verification only

### Static Analysis
- **Linting**: Run commands when possible, fix fixable errors

### Authorized Tools
- **Primary**: `fast_edit_block`, `fast_read_multiple_files`, filesystem-agent, ripgrep-agent
- **MCP Note**: Phase 4 tools not available in Continue.dev, use Kimi proxy endpoints
- **Complexity**: Major features require enhance_complex workflow

## Errors, Security, Cost

- **Errors**: Resolve introduced errors, revert if complex
- **Types**: No `any`, no degradation
- **Security/Cost**: Critical tasks for auth, DB, infrastructure, pricing

## Output Style

- **Lightweight**: 1-2 sentences
- **Standard+**: Headings, bullets, file references
- **Code**: Minimal examples, cite sources
- **User Priority**: Honor format/length specifications
